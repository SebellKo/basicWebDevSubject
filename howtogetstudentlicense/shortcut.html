IntelliJ ShortCut

  • - Alt + Enter

    1. 코드에서 오류가 발생, 코드 축소화, 툴에서 추천하는 제안으로 변경시에 코드위에 커서를 올려두고 alt + enter

  • - F2

    1. 수정이 필요한곳으로 바로 이동

  • - MacOS : CMD + 1, Win/Linux : Alt + 1

    1. 프로젝트창 열기

  • - ESC

    1. 에디터로 초점 옮기기

  • - MacOS : CMD + E, Win/Linux : Ctrl + E

    1. 가장 최근 열었던 파일을 확인

  • - MacOS : CMD + B, Win/Linux : Ctrl + B

    1. 코드안에서 다른 코드로 이동
    2. 클래스 이름에서 누르면 클래스 파일로 이동한다

  • - Alt + F7

    1. 인터페이스 이름에서 누를시 필드 선언 또는 인터페이스를 구현한 클래스를 비롯하여 해당 인터페이스가 사용된 모든 위치가 검색창에 표시된다.

  • - Ctrl twice

    1. 어떤 상태 어디에서든지 컨트롤을 두번 누르면 run anything창이 열린다 2. 기본적으로 run anything창에서는 최근 실행한 실행 구성 목록이 표시된다 3. 다른 실행 구성을 검색하려면 실행할 항목의 이름을 입력할 수도 있다

  • - MacOS : Option + upkey / Option + downkey, Win/Linux : Ctrl + W / Shift + Ctrl + W

    1. 커서 주변의 코드 섹션을 확장하거나 축소할 수 있다

  • - MacOS : CMD + /, Win/Linux : Ctrl + /

    1. 주석처리
    2. 블록 주석 처리시 alt + cmd + / 또는 Shift + Ctrl + /

  • - MacOS : Shift + CMD + Enter, Win/Linux : Shift + Ctrl + Enter

    1. 현재 구문 자동 완성 기능

  • - MacOS : Atl + CMD + L, Win/Linux : Ctrl + Alt + L

    1. 현재 파일의 서식을 프로젝트 표준에 맞추기

  • - MacOS : Ctrl + T, Win/Linux : Shift + Ctrl + Alt + T

    1. 모든 리팩토링을 표시
    2. 심볼이나 선택영역에서 이 단축키를 누르면 적용 가능한 리팩토링 옵션이 표시된다

  • - MacOS : Shift + CMD + A, Win/Linux : Shift + Ctrl + A

    1. 인텔리제이 모든 액션 검색

  • - Shift twice

    1. 무엇이든 검색 가능한 검색창 표시
